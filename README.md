# Metaphor Generator API (Beta)

日常の「モヤモヤ」を、最新の生成AI（Gemini 2.5 Flash）を用いて鮮烈な比喩表現へと変換し、クラウドへ自動保存するシステムです。

## 🚀 概要
本プロジェクトは、文学的な感性とクラウドエンジニアリングの技術的バックグラウンドを融合させる試みとして開発されました。
ユーザーの抽象的なストレスを、特定の「単語の衝突」ロジックに基づいた比喩へと錬成し、その結果を AWS S3 バケットへ永続化します。

**現在、本プロジェクトは「ベータ版」として公開されており、コアロジックの検証とインフラ連携を優先して実装しています。**

## 🛠 使用技術
- **Language:** Python 3.14
- **AI Model:** Google Gemini 2.5 Flash
- **Infrastructure:** AWS (S3)
- **Security:** `.env` による環境変数管理（GitHub公開時の認証情報漏洩防止済み）

## ✨ ベータ版の主要機能
- [x] **Gemini 2.5 連携**: 独自のシステムプロンプトによる比喩錬成ロジック。
- [x] **AWS S3 自動保存**: 生成結果を `output.txt` として指定のバケットへ自動アップロード。
- [x] **セキュリティ対策**: `python-dotenv` を用いた API キーの秘匿化。
